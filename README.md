# 🚗 CAR LANDING PAGE

A sleek and modern **car landing page** built using **HTML**, **CSS**, and **React.js**, designed to showcase automotive features with a clean user interface and responsive layout.

## 📌 Project Overview

This project focuses on creating a visually appealing landing page for **Nissan cars**. It demonstrates key frontend development skills, including responsive design, component structuring, and styling with CSS.

## 🛠️ Technologies Used

- 🔹 HTML5  
- 🔹 CSS3  
- 🔹 React.js

## ✨ Features

- 🔹 Responsive and mobile-friendly design  
- 🔹 Dynamic and interactive UI components  
- 🔹 Smooth scrolling and layout transitions  
- 🔹 Clean, well-structured code and reusable components
